## El sistema en una frase

**GBP en el centro. La web, las reseñas, el E-E-A-T y las redes sociales son los satélites que lo empujan al top 3.**

El diagnóstico siempre lo da el rank map (Leadsnap). La acción siempre sigue los 3 supuestos:
- Mapa rojo → Fase 1: Core 30
- Huevo frito (verde centro, bordes rojos) → Fase 3: Landmarks geográficos
- Rojo persistente tras Core 30 → Fase 2: Relevancia topical (PAA)

## Estructura del sistema (nuestro índice)

```
1. NUESTRO OBJETIVO → GBP
2. CÓMO IDENTIFICAMOS QUÉ HACER → Auditoría (Rank Map, 3 Supuestos, Toma de Decisiones)
3. CÓMO IMPACTAMOS EL GBP
   ├── GBP (Verificación, Descripción, Servicios, Categorías, Posts...)
   ├── WEB (Core 30, Arquitectura, Schema, Links, Lovable, Auditoría Técnica...)
   ├── RESEÑAS
   ├── E-E-A-T
   ├── REDES SOCIALES
   └── OPTIMIZACIÓN PARA IA (ChatGPT, Super Citations, Reddit...)
4. CÓMO GESTIONAMOS EL SISTEMA (Onboarding, Tracking, Producción, Mantenimiento)
5. NON-LOCAL SEO
```
